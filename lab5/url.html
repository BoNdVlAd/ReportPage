Опис схеми URL

Приклад :
<a href="https://www.example.com/page.html">Посилання на сторінку</a>

Протокол: Наприклад, http://, https://, ftp:// і т. д. Вказує, який протокол використовується для доступу до ресурсу.

Домен: Назва веб-сайту або IP-адреса сервера, який має ресурс. Наприклад, example.com або 192.168.1.1.

Порт (необов'язковий): Це номер порту, який використовується для з'єднання з веб-сервером. За замовчуванням, HTTP використовує порт 80, а HTTPS - 443.

Шлях: Це частина URL, яка вказує на конкретний ресурс на веб-сервері. Наприклад, /index.html, /products, /images/logo.png.

Параметри (необов'язкові): Додаткові параметри запиту, які передаються на сервер. Наприклад, ?id=123&category=books.

Якір (анкор): Це внутрішнє посилання на конкретну частину сторінки. Наприклад, #section1.

Запит (необов'язковий): Частина URL, яка містить додаткову інформацію для запиту, наприклад, пошуковий запит.
